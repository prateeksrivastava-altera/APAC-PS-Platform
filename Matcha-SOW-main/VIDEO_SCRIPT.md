## TIPS FOR RECORDING

### For Narration:
1. Record in a quiet environment
2. Use a quality microphone
3. Record multiple takes of each section
4. Add 1 second of silence before and after each section
5. Save each scene separately for easier editing

### For Screen Recording:
1. Use 1920x1080 resolution minimum
2. Hide unnecessary browser UI elements
3. Prepare sample data in advance (realistic but not confidential)
4. Practice cursor movements for smooth recording
5. Use keyboard shortcuts to avoid menu fumbling
6. Record at 60fps for smooth transitions

### For Editing:
1. Add subtle zoom effects to highlight important UI elements
2. Use fade transitions between major scenes
3. Keep text overlays on screen for 3-5 seconds
4. Synchronize visuals precisely with narration
5. Add sound effects for button clicks (subtle)
6. Include closed captions/subtitles for accessibility
